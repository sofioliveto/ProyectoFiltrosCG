# ProyectoCompuGrafica

Este proyecto es una aplicación de procesamiento de imágenes que permite aplicar diferentes filtros visuales.

## Cómo ejecutar la aplicación

1. Clona el repositorio:
    ```bash
    git clone <URL-del-repositorio>
    ```
2. Instala las dependencias necesarias (si aplica):
    ```bash
    npm install
    ```
3. Ejecuta cada filtro individualmente:
    ```bash
    python <nombre-del-archivo>
    ```
    o sigue las instrucciones específicas de tu entorno.

## Filtros disponibles

La aplicación incluye los siguientes filtros:

- **Filtro de escala de grises: vintage_cam vintage_filter** 
Convierte la imagen a tonos de gris, eliminando el color. Se encuentra la versión para aplicar a una foto fija y para aplicar a la cámara en vivo.

- **Filtro de accesorios: overlay_accesories** 
Agrega a un rostro que detecta en la cámara en vivo un sombrero, gafas y un bigote.

- **Filtro de cara graciosa: funny_face** 
Distorsiona una imágen con dos tipos de efectos diferentes.
